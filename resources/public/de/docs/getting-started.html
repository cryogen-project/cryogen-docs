Anfangen

Cryogens Ziel ist es, so einfach wie möglich zu sein. Sie müssen keine Datenbank aufsetzen oder viel Aufwand betreiben, nur um die Vorlage funktionsfähig zu machen.

Um zu beginnen, müssen Sie Leiningen installiert haben. Danach kommen Sie so zur Hauptvorlage:

~ $ lein new cryogen mein-blog
~ $ cd mein-blog
~/mein-blog $ lein ring server

Wenn der Server started, können Sie Ihre Seite auf localhost:3000 besuchen. Das erste, was Sie sehen, ist eine hilfreiche README-Datei, was Sie als nächstes tun sollten.
